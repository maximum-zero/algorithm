## 오늘 푼 문제

### 중성화 여부 파악하기 (프로그래머스)
- 유형: CASE WHEN (조건 분기)
- 핵심
    - CASE WHEN → 조건에 따라 값 변환
    - LIKE → 문자열 패턴 매칭
- 실수: 없음
- 배운 점
    - SQL에서도 if문처럼 조건 분기 가능

[풀이]
```sql
SELECT
    ANIMAL_ID,
    NAME,
    CASE
        WHEN SEX_UPON_INTAKE LIKE 'Neutered%' OR SEX_UPON_INTAKE LIKE 'Spayed%' THEN 'O'
        ELSE 'X'
    END AS 중성화
FROM ANIMAL_INS;
```

### DATETIME에서 DATE로 형 변환 (프로그래머스)
- 유형: 날짜 함수 (DATE_FORMAT)
- 핵심
  - DATE_FORMAT → 날짜 형식 변환
  - '%Y-%m-%d' → 연-월-일
- 실수: 없음
- 배운 점: 날짜 데이터는 문자열로 변환해서 출력 가능

[풀이]
```sql
SELECT
    ANIMAL_ID,
    NAME,
    DATE_FORMAT(DATETIME, '%Y-%m-%d') AS 날짜
FROM ANIMAL_INS
ORDER BY ANIMAL_ID;
```

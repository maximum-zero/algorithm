## 오늘 푼 문제

### [프로그래머스 - 오랜 기간 보호한 동물(2)](https://school.programmers.co.kr/learn/courses/30/lessons/59411)
- 유형: JOIN + 날짜 차이 계산
- 핵심 
  - 보호 기간 = OUTS - INS 
  - TIMESTAMPDIFF로 시간 차이 계산
  - 보호 기간 기준 내림차순 정렬 후 상위 2개
- 실수
  - 시간 차이 계산 순서를 반대로 작성함
- 배운 점
  - TIMESTAMPDIFF() 사용법

[풀이]
```sql
SELECT
    AOUTS.ANIMAL_ID,
    AOUTS.NAME
FROM ANIMAL_OUTS AOUTS
    INNER JOIN ANIMAL_INS AINS ON AINS.ANIMAL_ID = AOUTS.ANIMAL_ID
ORDER BY TIMESTAMPDIFF(SECOND, AINS.DATETIME, AOUTS.DATETIME) DESC
LIMIT 2;
```

### [프로그래머스 - 있었는데요 없었습니다](https://school.programmers.co.kr/learn/courses/30/lessons/59043)
- 유형: JOIN + 조건 필터링
- 핵심 
  - 입양 기록이 존재하는 데이터만 JOIN 
  - 입소일 > 입양일 → 이상 데이터 필터링
- 실수: 없음

[풀이]
```sql
SELECT
    AINS.ANIMAL_ID,
    AINS.NAME
FROM ANIMAL_INS AINS
    INNER JOIN ANIMAL_OUTS AOUTS ON AINS.ANIMAL_ID = AOUTS.ANIMAL_ID
WHERE AINS.DATETIME > AOUTS.DATETIME
ORDER BY AINS.DATETIME;
```

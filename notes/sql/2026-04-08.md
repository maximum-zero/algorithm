## 오늘 푼 문제

### 조건에 맞는 회원수 구하기 (프로그래머스)
- 유형: COUNT + WHERE
- 핵심
  - 나이 조건 (20~29)
  - 가입 연도 조건 (2021년)
- 실수: 나이 조건만 확인하고 가입 연도 조건 누락
- 원인: 문제 조건을 끝까지 읽지 않음
- 교정: WHERE 조건 전체 확인 후 쿼리 작성

[풀이]
```sql
SELECT
    COUNT(*)
FROM USER_INFO
WHERE AGE BETWEEN 20 AND 29
    AND YEAR(JOINED) = 2021;
```

### 이름이 없는 동물의 아이디 (프로그래머스)
- 유형: NULL 처리
- 핵심
  - NULL 값 조회 → IS NULL 사용
  - 정렬 필요 → ORDER BY
- 실수: 없음

[풀이]
```sql
SELECT
    ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NULL
ORDER BY ANIMAL_ID ASC;
```

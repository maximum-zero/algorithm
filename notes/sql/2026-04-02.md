## 문제 1
### 오랜 기간 보호한 동물(1) (프로그래머스)

[유형]
- LEFT JOIN + NULL 필터 + 정렬

[풀이]
```sql
SELECT
    AINS.NAME,
    AINS.DATETIME
FROM ANIMAL_INS AINS
LEFT JOIN ANIMAL_OUTS AOUTS
ON AINS.ANIMAL_ID = AOUTS.ANIMAL_ID
WHERE AOUTS.ANIMAL_ID IS NULL
ORDER BY AINS.DATETIME
LIMIT 3;
```

[실수]
- LEFT JOIN 후 NULL 필터 누락

[핵심]
- LEFT JOIN + IS NULL → "존재하지 않는 데이터 찾기"


## 문제2
### 상품 별  오프라인 매출 구하기 (프로그래머스)

[유형]
- JOIN + GROUP BY + 집계 함수

[풀이]
```sql
SELECT
    P.PRODUCT_CODE,
    SUM(OS.SALES_AMOUNT * P.PRICE) AS SALES
FROM PRODUCT P
INNER JOIN OFFLINE_SALE OS
ON OS.PRODUCT_ID = P.PRODUCT_ID
GROUP BY P.PRODUCT_CODE
ORDER BY SALES DESC, P.PRODUCT_CODE ASC;
```

[실수]
- GROUP BY 기준 혼동

[핵심]
- GROUP BY → SELECT 기준으로 묶기

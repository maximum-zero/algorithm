## 오늘 푼 문제

### 중복 제거하기 (프로그래머스)
- 유형: DISTINCT + COUNT
- 핵심
  - DISTINCT: 중복 제거
  - COUNT: 개수 집계
  - NULL 제외 필요
- 실수: 없음

[풀이]
```sql
SELECT
  COUNT(DISTINCT NAME) AS count
FROM ANIMAL_INS
WHERE NAME IS NOT NULL;
```

### 이름이 있는 동물의 아이디 (프로그래머스)
- 유형: NOT NULL + 정렬
- 핵심
  - IS NOT NULL: NULL 제외 
  - ORDER BY: 결과 정렬
- 실수: 없음

[풀이]
```sql
SELECT
  ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
ORDER BY ANIMAL_ID ASC;
```

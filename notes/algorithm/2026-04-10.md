# 📅 2026-04-10

## 오늘 푼 문제

### [BOJ 10815 - 숫자 카드](https://www.acmicpc.net/problem/10815)
- 유형: Set (존재 여부)
- 핵심
  - contains() → O(1) 조회
  - 반복 질의 처리
- 실수: 교집합으로 착각
- 원인: Set 문제를 하나로 묶어서 생각
- 교정: "존재 여부 / 교집합 / 차집합" 구분

### [BOJ 1269 - 대칭 차집합](https://www.acmicpc.net/problem/1269)
- 유형: Set (대칭 차집합)
- 핵심
  - A - B + B - A
  - removeAll()으로 직관적 구현
- 실수: 없음
- 배운 점: Set 연산으로 문제 의도를 그대로 표현 가능

---

## 🧠 오늘 배운 패턴
- Set
  - contains → 존재 여부 
  - 교집합 → 둘 다 포함 
  - 차집합 → 한쪽에만 포함

---

## ⚠️ 오늘 인사이트
- 같은 Set이라도 문제 유형이 다름
  - 존재 여부 
  - 교집합 
  - 차집합

👉 문제를 “연산 기준”으로 구분해야 함

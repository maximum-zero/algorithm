# 📅 2026-04-13

## 오늘 푼 문제

### [BOJ 11652 - 카드](https://www.acmicpc.net/problem/11652)
- 유형: HashMap (빈도수 카운팅)
- 핵심
  - 숫자 → 등장 횟수 저장
  - 최댓값 갱신 + 같은 경우 더 작은 값 선택
- 실수: 없음

### [BOJ 1927 - 최소 힙](https://www.acmicpc.net/problem/1927)
- 유형: PriorityQueue (heap)
- 핵심
  - offer() → 삽입
  - poll() → 최솟값 추출
  - 비어있으면 0 출력
- 실수: 없음
- 배운 점
  - "최솟값 반복 추출" → PriorityQueue

---

## 🧠 오늘 배운 패턴
- Map
  - 빈도수 카운팅
  - 최댓값 갱신

- Set
  - 교집합 처리
  - contains() 활용

- PriorityQueue
  - 최소값 / 최대값 관리
  - 자동 정렬 구조

---

## ⚠️ 오늘 인사이트
- 같은 자료구조라도 “사용 목적”이 다름
  - Map → 카운팅
  - Set → 존재 / 집합 연산
  - PQ → 우선순위
